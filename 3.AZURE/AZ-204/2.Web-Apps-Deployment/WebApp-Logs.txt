Application Logging: log messages that are generated by your application code.
Web Server Logging: raw HTTP request data.
Detailed Error Messages: copies of .htm error pages that would have been sent to the client browser.
Deployment Logging: logs when you publish content to an application.


1.Web App'te Application ve Web Server Loglarin Konfigurasyonu:
> webapp1 > App service logs
  > Application loging: On (Filesyetem) | Level: Information
    - web app'i tutan ec2'da tutulur.
    - Burayi "On" yaparsak blob'u off yapariz.
  > Application logging (Blob): Off
  > Web server logging: File system
    - Storage dersek blob'da File system dersek web app'a host'luk yapan ec2'da tutulur.
  > Save



2.Web App'te Log stream Konfigurasyonu:
- Once Application logs, Web Server logs konfigure edilmeli.
> webapp1 > Log stream
  > Application logs  - check
    - Application loglarinin real time gorebilir.
  > Web Server logs
    - Web Server logs'lari real time gorur.
